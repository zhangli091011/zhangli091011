## Things I Built

### 校园内网文件分发系统
因为经常要拿着 U 盘给全年级很多班级拷文件，我决定不再靠双腿解决问题。  
于是做了一个基于内网分发 / P2P 思路的文件分发系统，现在已经在校内推广，可以在 **1 分钟内把文件分发到近 100 个班级**。

### DCR 学生互助平台
为了替代频繁被风控的原有渠道，团队开始开发新的学生互助网站。  
项目支持 **三方互助链路、站内联系、工单全流程跟踪、权限分级、后台审核、自动生成委托表**，目标是让互助这件事更安全、更稳定、更可持续。

### 彩色二维码文件传输方案
一个有点“离谱但真在做”的项目。  
通过 **彩色多通道并行承载 + 纠删码** 提高二维码文件传输带宽，并加入断点续传、缺块补传、进度显示、校验和纠错等机制。目前已完成交底书撰写，正在推进专利申请。

### 校园活动一站式服务平台
参与并开发了校园活动相关的一站式平台，整合了 **报名、审核、反馈、投稿、新闻更新** 等功能，让活动组织不再靠表格满天飞。
